Web Usability

  1. Different font sizes, weights, and colors are used in order to increase the readablity.
  2. There is a reasonable amount of white space between different sections of the page.
  3. There is a navigation bar in every page that gives the user easy access to all pages.
  4. The anvigation bar stays at the top of the page as the user scrolls down so that the user can access the navigation bar no matter where they are.
  5. There is enough space between letters and lines of text.
  6. Labs and Assignments in the navigation bar are drop down menus and user can access all labs and assignments through them. Also, they're accessible in every page meaning the user can open each link by just one click no matter which page they are currently at.
  7. As mentioned before, Labs and Assignments are drop down menus. Not only it organizes the content, but it also prevents the page from being overloaded with data.
  8. The navigation bar is responsive and it changes its layout depending on the size of the screen.
  9. Pages have a grid layout which organizes the page and they're responsive. They change their layout depending on the size of the screen.
  10. All the pages follow the same designing pattern which makes it consistent.
  11. The design is familiar for the user. For example the icon used for the drop-down menu has been already used in many other website for the same purpose.
